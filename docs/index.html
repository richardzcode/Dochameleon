    Progressive Customization

    Works out-of-box, able to customize to the extend of replacing core components

    Quick Setup

    Get up and running quickly. Have your own website in just a few minutes

    > npm install -g dochameleon-init
> dochameleon-init
> cd website
> npm run start
    Develop and Deploy

    Develop using included live server; Deploy to GitHub or any static file web hosts
